# Cost Explorer

## 개요
AWS Cost Explorer는 AWS 비용과 사용량을 시각화하고 분석할 수 있는 도구입니다. 시간에 따른 비용 추세를 파악하고 비용 최적화 기회를 찾을 수 있습니다.

## 핵심 개념

### 비용 분석 기능

#### 1. 비용 및 사용량 보고서
- **시간별/일별/월별** 비용 추세
- **서비스별** 비용 분석
- **리전별** 비용 분석
- **계정별** 비용 분석 (조직)

#### 2. 예측 (Forecasting)
- 향후 비용 예측
- 머신러닝 기반 예측
- 예산 계획 수립에 활용

#### 3. 예약 인스턴스 권장 사항
- 예약 인스턴스 구매 권장 사항
- 예상 절감액 표시
- 사용 패턴 분석

### 비용 할당 태그 (Cost Allocation Tags)
- **사용자 정의 태그**: 리소스에 태그 추가
- **AWS 생성 태그**: 자동 생성 태그
- **태그 기반 필터링**: 태그별 비용 분석
- **태그 정책**: 태그 사용 강제

## 주요 기능

### 1. 비용 대시보드
- 총 비용 요약
- 서비스별 비용
- 리전별 비용
- 시간별 추세

### 2. 사용량 분석
- 리소스별 사용량
- 시간대별 사용 패턴
- 피크 사용 시간 식별

### 3. 비용 최적화 권장 사항
- 미사용 리소스 식별
- 예약 인스턴스 권장
- Savings Plans 권장

### 4. 예산 설정
- 비용 예산 설정
- 사용량 예산 설정
- 예산 초과 시 알림

## 사용 사례

### 1. 비용 모니터링
- 월별 비용 추세 파악
- 예상치 못한 비용 증가 감지
- 서비스별 비용 분포 확인

### 2. 비용 최적화
- 미사용 리소스 식별
- 예약 인스턴스 구매 결정
- 스토리지 클래스 최적화

### 3. 예산 관리
- 부서별 예산 할당
- 프로젝트별 비용 추적
- 비용 예측 및 계획

## 시험 포인트

1. **비용 할당 태그**
   - 태그 활성화 후 비용 데이터 수집 시작
   - 태그는 과거 비용에 적용되지 않음
   - 태그 기반 필터링 및 그룹화

2. **예약 인스턴스 권장**
   - 사용 패턴 분석 기반
   - 예상 절감액 표시
   - 구매 권장 사항 제공

3. **비용 예측**
   - 과거 데이터 기반 예측
   - 계절성 고려
   - 예산 계획 수립에 활용

4. **리소스 식별**
   - 태그를 통한 리소스 그룹화
   - 미사용 리소스 식별
   - 비용 드라이버 분석

5. **조직 단위 분석**
   - 여러 계정 비용 통합
   - 계정별 비용 분리
   - 조직 전체 비용 관리

## 모범 사례

1. **태그 전략**
   - 일관된 태그 명명 규칙
   - 필수 태그 정의 (예: Environment, Project, Owner)
   - 태그 정책으로 강제

2. **정기적 검토**
   - 주간/월간 비용 검토
   - 예상치 못한 비용 증가 조사
   - 비용 최적화 기회 탐색

3. **예산 설정**
   - 서비스별 예산 설정
   - 부서/프로젝트별 예산
   - 예산 초과 알림 구성

4. **비용 할당**
   - 태그를 통한 비용 할당
   - 부서/팀별 비용 추적
   - 프로젝트별 비용 관리

5. **리포트 활용**
   - 정기적인 비용 리포트 생성
   - 이해관계자에게 공유
   - 의사결정에 활용

## 비용 분석 예시

### 서비스별 비용
```
EC2: 40%
RDS: 25%
S3: 15%
Data Transfer: 10%
기타: 10%
```

### 리전별 비용
```
us-east-1: 60%
ap-northeast-2: 30%
eu-west-1: 10%
```

## 관련 서비스
- **AWS Budgets**: 예산 설정 및 알림
- **Cost and Usage Report**: 상세 비용 데이터
- **AWS Organizations**: 다중 계정 비용 관리
- **Trusted Advisor**: 비용 최적화 권장 사항
